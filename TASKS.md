## User Onboarding & 10DLC

### 10DLC Signup Flow
When user creates account and signs in for the first time:

1. **Get Verified Screen**
   - Explain why verification is needed
   - Show benefits of verification
   - CTA: "Start Verification"

2. **Business Information**
   - Business name
   - Business type (LLC, Corp, Sole Prop, etc.)
   - EIN (optional for Sole Prop)
   - Business address
   - Website URL

3. **Use Case Declaration**
   - Select messaging use case (Marketing, Notifications, 2FA, etc.)
   - Describe message content
   - Estimated monthly volume
   - Sample messages

4. **Phone Number Selection**
   - Search by area code
   - Show available numbers
   - Select and purchase

5. **Verification Submitted**
   - Show pending status
   - Explain review timeline (24-48 hours)
   - What they can do while waiting

### Post-Verification
- Dashboard access unlocked
- Phone number activated
- Ready to send messages
