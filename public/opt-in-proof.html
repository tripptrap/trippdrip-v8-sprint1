SMS Compliance Documentation

HyveWyre SMS Opt-In Workflow

Documentation of consent collection process for toll-free messaging verification

1. Business Overview

HyveWyre LLC is an Independent Software Vendor (ISV) providing a SaaS CRM platform to licensed insurance agents and real estate professionals. Each client business that subscribes to our platform is assigned their own dedicated toll-free number for customer communications.

Message Types

Appointment reminders, policy renewal notifications, and follow-up messages to customers who have provided prior express written consent through the opt-in workflow described below.

2. Opt-In Collection Method

Customers provide SMS consent through web-based contact forms on agent websites. The consent mechanism requires an affirmative action (checking an unchecked checkbox) with clear disclosure language.

3. Consent Workflow Steps

1

Customer Visits Agent Website

Prospective customer navigates to the agent's contact or quote request page hosted on the HyveWyre platform.

2

Form Completion with SMS Disclosure

Customer fills out required fields and views the SMS consent checkbox. The checkbox is unchecked by default and requires affirmative action to enable SMS communications.

3

Explicit Consent Provided

Customer checks the SMS consent box, acknowledging the disclosure language that includes: message frequency, data rates, STOP/HELP keywords, and links to Privacy Policy and Terms.

4

Consent Logged with Timestamp

Upon form submission, consent is recorded in our database with timestamp, phone number, source URL, and IP address for compliance records.

5. Opt-Out Mechanism

All messages include opt-out instructions. The platform automatically processes the following keywords:

  • STOP - Immediately unsubscribes the recipient and adds to Do-Not-Call list
  • HELP - Sends assistance information with support contact details
  • START / YES - Re-subscribes a previously opted-out number (if requested)

Automatic Opt-Out Response

"You have been unsubscribed and will no longer receive messages. Reply START to resubscribe."

6. First Message Compliance

The first message sent to any new recipient automatically includes opt-out instructions appended to the message body:

"Hi John, this is Sarah from ABC Insurance. Following up on your quote request. Let me know if you have any questions!

Reply STOP to opt out"

7. Platform-Level Enforcement

HyveWyre enforces consent requirements at the platform level:

  • Contacts cannot receive messages unless SMS consent is recorded
  • Do-Not-Call list is maintained and checked before every message
  • STOP keyword processing is automatic and immediate
  • Consent records are retained for compliance auditing
  • Agents cannot bypass consent requirements
